# Stellar 
 Stellar is code-review tool which reviews your code and help you to find bugs, suggest improvments and optimize your code. It will also provide a score based on your's code quality and rewrite the optimized version of your code.

# Tech Stacks

- Next JS(App Router) --> Frontend + API routes
- Gemini API --> Integrated `Gemini API` in the API routes and then connected it to the Frontend
- Framer Motion --> To add animations in the UI
- Tailwind CSS
- Typescript

# Deployment
Deployed on Vercel

# Run Locally

- Fork the repo
- Clone the repo
```bash
  git clone forked_repo_url
```
- Install Dependencies
```bash
 cd stellar
 npm install
```

- Run the server
```bash
 npm run dev
```
